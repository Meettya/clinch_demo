Greater Example

Пример функции, использующей шаблон jade

Подгрузив на страницу собранный с помощью clinch бандл, можно извлечь из него нужную нам функцию так:

{greater} = clinch_demo.main

Далее просто используем ее для вывода информации 

greater name : 'Bender'

Эта функция даст нам вот такой результат (сформировано на клиенте)
